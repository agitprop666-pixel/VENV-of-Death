

VENV of Death is a virtual environment manager aimed at Python coders.
Less tweaking over the command line = More Productivity.
Run as root/admin if the scripts you're running need elevation.
Features

    Create new venv's.
    Run different venv's.
    Install, delete, or upgrade PIP packages.
    Lists installed packages.
    Run Python scripts, drag and drop.
    It has it's own console.
    Source code will run on Windows, should run on Mac as well.
